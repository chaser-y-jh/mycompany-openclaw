# Northflank

Deploy MerClaw on Northflank with a one-click template and access it through the web Control UI.
This is the easiest "no terminal on the server" path: Northflank runs the Gateway for you.

## How to get started

1. Click [Deploy MerClaw](https://northflank.com/stacks/deploy-merclaw) to open the template.
2. Create an [account on Northflank](https://app.northflank.com/signup) if you don't already have one.
3. Click **Deploy MerClaw now**.
4. Set the required environment variable: `MERCLAW_GATEWAY_TOKEN` (use a strong random value).
5. Click **Deploy stack** to build and run the MerClaw template.
6. Wait for the deployment to complete, then click **View resources**.
7. Open the MerClaw service.
8. Open the public MerClaw URL at `/merclaw` and connect using the configured shared secret. This template uses `MERCLAW_GATEWAY_TOKEN` by default; if you replace it with password auth, use that password instead.

## What you get

- Hosted MerClaw Gateway + Control UI
- Persistent storage via Northflank Volume (`/data`) so `merclaw.json`,
  per-agent `auth-profiles.json`, channel/provider state, sessions, and
  workspace survive redeploys

## Connect a channel

Use the Control UI at `/merclaw` or run `merclaw onboard` via SSH for channel setup instructions:
