# 考勤打卡管理

## 功能

协助教师完成每日考勤打卡，包括：
- 全班点名（批量录入）
- 单个学生考勤修正
- 查看班级出勤汇总
- 查看学生月度出勤统计
- 查看未打卡学生名单

## 使用方式

教师对 Agent 说：
- "今天三年级二班点名，到勤30人，张三迟到，李四请假"
- "查看三年级二班 2026-06-04 的考勤"
- "王五这个月出勤率多少"
- "还有谁没打卡"

学生/家长对 Agent 说：
- "我这个月出勤情况怎么样"

## 考勤状态说明

| 状态 | 中文 | 说明 |
|------|------|------|
| present | 到勤 | 准时到校 |
| late | 迟到 | 迟到但已到 |
| absent | 缺勤 | 未到且未请假 |
| leave | 请假 | 已请假批准 |

## 出勤率计算

出勤率 = (到勤天数 + 迟到次数) / (到勤 + 迟到 + 缺勤) × 100%
请假不计入分母（不影响出勤率）。

## 权限要求

- 记录考勤：教师、管理员（attendance:record）
- 查看班级考勤：教师、管理员（attendance:view_class）
- 查看个人考勤：学生、教师、家长（attendance:view_own）
