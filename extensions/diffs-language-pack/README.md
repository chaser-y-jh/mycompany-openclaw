# @merclaw/diffs-language-pack

Official extended syntax highlighting pack for the MerClaw Diffs plugin.

The base `@merclaw/diffs` plugin ships a curated language set. Install this package when you want the full Shiki language catalog available in rendered diff viewers and diff image/PDF output.

## Install

```bash
merclaw plugins install @merclaw/diffs-language-pack
```

Restart the Gateway after installing or updating the plugin.

## Use with Diffs

Install `@merclaw/diffs` first, then install this language pack. The language pack contributes static viewer assets; it does not register a separate agent tool.
